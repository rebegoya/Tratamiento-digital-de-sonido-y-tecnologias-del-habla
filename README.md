# Tratamiento-digital-de-sonido-y-tecnologias-del-habla
En este repositorio se incluyen las practicas de la asignatura Tratamiento Digital de Sonido del Grado en Ingeniería en Sistemas Audiovisuales y Multimedia de la URJC.
Contiene Jupiter notebooks, ficheros de código adicional y muestras de señales de audio.
También se incluye la definición de los proyectos que los/as estudiantes deben realizar en la asingatura.
